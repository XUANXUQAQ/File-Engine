Welcome to the tutorial of File-Engine=Benvenuti al tutorial di File-Engine
The default Ctrl + Alt + K calls out the search bar, which can be changed in the settings.=Il valore predefinito Ctrl + Alt + K richiama la barra di ricerca, che può essere cambiata nelle impostazioni.
You can enter the keywords you want to search here=Puoi digitare le parole chiave che vuoi cercare qui
Let's see an example=Vediamo un esempio.
When you enter "test" in the search bar=Quando si digita "prova" nella barra di ricerca
files with "test" in the name will be displayed below the search bar=I file con "prova" nel nome saranno visualizzati sotto la barra di ricerca
If you know multiple keywords of a file=Se conosci più parole chiave di un file
(for example, the file name contains both "file" and "test")=(ad esempio, il nome del file contiene sia "file" che "test")
you can separate them with ";" (semicolon) to search together as keywords.=è possibile separarli con ";" (punto e virgola) per cercare insieme come parole chiave.
When entering "/test" in the search bar=Quando inserisci "/prova" nella barra di ricerca
the file containing "test" in the path will be displayed below the search bar=Il file contenente "prova" nel percorso sarà visualizzato sotto la barra di ricerca
Add "::" + suffix after the keyword to achieve a more precise search=Aggiungi "::" + suffisso dopo la parola chiave per ottenere una ricerca più precisa
The program has the following four suffixes=Il programma ha i seguenti quattro suffissi
not case sensitive=non caso sensibile
"::d" is the suffix for searching only folders="::d" è il suffisso per la ricerca solo nelle cartelle
"::f" is the suffix to search only for files="::f" è il suffisso per cercare solo i file
"::full" means full word matching, but case insensitive="::full" significa corrispondenza di parole complete, ma senza distinzione tra maiuscole e minuscole
"::case" means case sensitive="::case" significa case sensitive
You can also combine different suffixes to use=È inoltre possibile combinare diversi suffissi da usare
Different keywords are separated by ";" (semicolon), suffix and keywords are separated by "::" (colon)=Le parole chiave diverse sono separate da ";" (punto e virgola), suffisso e parole chiave sono separate da "::" (colon)

Click "Shift + Enter" to open the file as an administrator (use with caution)=Fare clic su "Shift + Enter" per aprire il file come amministratore (usare con cautela)
Click "Alt+ Enter" to copy the file path=Fare clic su "Alt+ Enter" per copiare il percorso del file
You can modify these hotkeys in the settings=È possibile modificare questi tasti hotkey nelle impostazioni
Enter ":" (colon) at the front of the search box to enter the command mode=Inserisci "(colon) nella parte anteriore della casella di ricerca per inserire la modalità di comando
There are built-in commands, you can also add custom commands in the settings=Ci sono comandi incorporati, è anche possibile aggiungere comandi personalizzati nelle impostazioni
If you find that some files cannot be searched, you can enter ":update" in the search bar to rebuild the index.=Se trovi che alcuni file non possono essere consultati, puoi inserire ":update" nella barra di ricerca per ricostruire l'indice.
