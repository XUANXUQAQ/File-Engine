Welcome to the tutorial of File-Engine=ファイルエンジンのチュートリアルへようこそ
The default Ctrl + Alt + K calls out the search bar, which can be changed in the settings.=デフォルトのCtrl + Alt + kは設定で変更できる検索バーを呼び出します。
You can enter the keywords you want to search here=ここで検索するキーワードを入力できます
Let's see an example=例を見ましょう
When you enter "test" in the search bar=検索バーで「test」を入力すると
files with "test" in the name will be displayed below the search bar=名前の「テスト」ファイルが検索バーの下に表示されます
If you know multiple keywords of a file=あなたがファイルの複数のキーワードを知っているならば
(for example, the file name contains both "file" and "test")=(例えば、ファイル名と" test "の両方が含まれます。
you can separate them with ";" (semicolon) to search together as keywords.=あなたは"；（セミコロン）をキーワードとして一緒に検索するとそれらを分離することができます。
When entering "/test" in the search bar=検索バーに「/ test」を入力すると
the file containing "test" in the path will be displayed below the search bar=パス内の“test”を含むファイルが検索バーの下に表示されます
Add ":" + suffix after the keyword to achieve a more precise search=より正確な検索を実現するには、キーワードの後に「:」+サフィックスを追加します
The program has the following four suffixes=プログラムは以下の4つの接尾辞を持っている
not case sensitive=大文字小文字を区別しない
":d" is the suffix for searching only folders=「:d」はフォルダのみを検索するためのサフィックスです
":f" is the suffix to search only for files=「:f」は、ファイルのみを検索するためのサフィックスです
":full" means full word matching, but case insensitive=「:full」は完全な単語の一致を意味しますが、大文字と小文字は区別されません
":case" means case sensitive=「:case」は大文字と小文字を区別することを意味します
You can also combine different suffixes to use=また、使用する別の接尾辞を組み合わせることができます
Different keywords are separated by ";" (semicolon), suffix and keywords are separated by ":" (colon)=異なるキーワードは「;」で区切られます （セミコロン）、サフィックス、キーワードは「:」（コロン）で区切られます

If you find that some files cannot be searched, you can enter ":update" in the search bar to rebuild the index.=いくつかのファイルを検索できない場合は、検索バーで「更新」を入力してインデックスを再構築します。
