Welcome to the tutorial of File-Engine=歡迎進入File-Engine的教程
The default Ctrl + Alt + K calls out the search bar, which can be changed in the settings.=默認Ctrl + Alt + K調出蒐索欄，可以在設定中進行更改。
You can enter the keywords you want to search here=您可以在此處輸入要蒐索的關鍵字
Let's see an example=讓我們看一個例子
When you enter "test" in the search bar=當您在蒐索欄中輸入“test”時
files with "test" in the name will be displayed below the search bar=名稱中帶有“test”的檔案將顯示在蒐索欄下方
If you know multiple keywords of a file=如果您知道一個檔案的多個關鍵字
(for example, the file name contains both "file" and "test")=（例如，檔名同時包含“file”和“test”）
you can separate them with ";" (semicolon) to search together as keywords.=您可以使用“；”（分號）將它們分開作為關鍵字進行蒐索。
When entering "/test" in the search bar=在蒐索欄中輸入“/ test”時
the file containing "test" in the path will be displayed below the search bar=路徑中包含“test”的檔案將顯示在蒐索欄下方
Add "::" + suffix after the keyword to achieve a more precise search=在關鍵字之後添加“::”+尾碼以實現更精確的蒐索
The program has the following four suffixes=該程式具有以下四個尾碼
not case sensitive=不區分大小寫
"::d" is the suffix for searching only folders=“::d”是僅蒐索資料夾的尾碼
"::f" is the suffix to search only for files=“::f”是僅蒐索檔案的尾碼
"::full" means full word matching, but case insensitive=“::full”表示全詞匹配，但不區分大小寫
"::case" means case sensitive=“::case”表示區分大小寫
You can also combine different suffixes to use=您還可以組合使用不同的尾碼
Different keywords are separated by ";" (semicolon), suffix and keywords are separated by "::" (colon)=不同的關鍵字用“；”（分號）分隔，尾碼和關鍵字用“::”（冒號）分隔
