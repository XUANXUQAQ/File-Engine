Welcome to the tutorial of File-Engine=파일 엔진 튜 토리 얼 에 오신 것 을 환영 합 니 다.
The default Ctrl + Alt + K calls out the search bar, which can be changed in the settings.=기본 Ctrl + Alt + K 에서 검색 창 을 바 꾸 면 설정 에서 변경 할 수 있 습 니 다.
You can enter the keywords you want to search here=검색 할 키 워드 를 입력 할 수 있 습 니 다
Let's see an example=예 를 보 여 주세요.
When you enter "test" in the search bar=검색 창 에 "test" 를 입력 할 때
files with "test" in the name will be displayed below the search bar=이름 에 "test" 가 있 는 파일 은 검색 표시 줄 아래 에 표 시 됩 니 다.
If you know multiple keywords of a file=파일 하나 에 여러 키 워드 를 알 고 있다 면
(for example, the file name contains both "file" and "test")=(예 를 들 어 파일 이름 은 'file' 과 'test' 를 동시에 포함 합 니 다)
you can separate them with ";" (semicolon) to search together as keywords.=";" (분점) 으로 나 누 어 키워드 로 함께 검색 할 수 있 습 니 다.
When entering "/test" in the search bar=검색 창 에 "/ 테스트" 를 입력 할 때
the file containing "test" in the path will be displayed below the search bar=경로 에 'test' 를 포함 하 는 파일 은 검색 표시 줄 아래 에 표 시 됩 니 다.
Add ":" + suffix after the keyword to achieve a more precise search=보다 정확한 검색을 위해 키워드 뒤에 ":" + 접미사 추가
The program has the following four suffixes=이 프로그램 은 아래 네 개의 접미사 가 있다.
not case sensitive=대소 문자 구분 안 함
":d" is the suffix for searching only folders=":d"는 폴더만 검색하기 위한 접미사입니다.
":f" is the suffix to search only for files=":f"는 파일만 검색하는 접미사입니다.
":full" means full word matching, but case insensitive=":full"은 전체 단어 일치를 의미하지만 대소문자를 구분하지 않습니다.
":case" means case sensitive=":case"는 대소문자를 구분한다는 의미입니다.
You can also combine different suffixes to use=다른 접 미 어 를 조합 해서 사용 할 수 있 습 니 다.
Different keywords are separated by ";" (semicolon), suffix and keywords are separated by ":" (colon)=다른 키워드는 ";"으로 구분됩니다. (세미콜론), 접미사 및 키워드는 ":"(콜론)으로 구분됩니다.
Click "Enter" to open the file directly="Enter" 를 누 르 면 바로 파일 을 연다.
Click "Ctrl + Enter" to open the folder where the file is located="Ctrl + Enter" 를 누 르 면 파일 이 있 는 폴 더 를 엽 니 다.
Click "Shift + Enter" to open the file as an administrator (use with caution)="Shift + Enter" 를 누 르 면 관리자 로 서 파일 을 엽 니 다 (조심해 서 사용)
Click "Alt+ Enter" to copy the file path="Alt + Enter" 를 누 르 면 파일 경 로 를 복사 합 니 다.
You can modify these hotkeys in the settings="설정" 에서 이 단축 키 를 수정 할 수 있 습 니 다.
Enter ":" (colon) at the front of the search box to enter the command mode=검색 창 앞 에 ":" (콜론) 을 입력 하여 명령 모드 에 들 어 갑 니 다.
There are built-in commands, you can also add custom commands in the settings=내 장 된 명령 이 있 으 면 설정 에 사용자 정의 명령 을 추가 할 수도 있 습 니 다.
If you find that some files cannot be searched, you can enter ":update" in the search bar to rebuild the index.=검색 할 수 없 는 파일 이 있 으 면 검색 창 에 "update" 를 입력 하여 색인 을 다시 만 들 수 있 습 니 다.
