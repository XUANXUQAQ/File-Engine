Welcome to the tutorial of File-Engine=欢迎进入File-Engine的教程
The default Ctrl + Alt + K calls out the search bar, which can be changed in the settings.=默认Ctrl + Alt + K调出搜索栏，可以在设置中进行更改。
You can enter the keywords you want to search here=您可以在此处输入要搜索的关键字
Let's see an example=让我们看一个例子
When you enter "test" in the search bar=当您在搜索栏中输入“test”时
files with "test" in the name will be displayed below the search bar=名称中带有“ test”的文件将显示在搜索栏下方
If you know multiple keywords of a file=如果您知道一个文件的多个关键字
(for example, the file name contains both "file" and "test")=（例如，文件名同时包含“file”和“test”）
you can separate them with ";" (semicolon) to search together as keywords.=您可以使用“;”（分号）将它们分开作为关键字进行搜索。
When entering "/test" in the search bar=在搜索栏中输入“ / test”时
the file containing "test" in the path will be displayed below the search bar=路径中包含“ test”的文件将显示在搜索栏下方
Add ":" + suffix after the keyword to achieve a more precise search=在关键字之后添加“:” +后缀以实现更精确的搜索
The program has the following four suffixes=该程序具有以下四个后缀
not case sensitive=不区分大小写
":d" is the suffix for searching only folders=“:d”是仅搜索文件夹的后缀
":f" is the suffix to search only for files=“:f”是仅搜索文件的后缀
":full" means full word matching, but case insensitive=“:full”表示全字匹配，但不区分大小写
":case" means case sensitive=“:case”表示区分大小写
You can also combine different suffixes to use=您还可以结合使用不同的后缀
Different keywords are separated by ";" (semicolon), suffix and keywords are separated by ":" (colon)=不同的关键字用“；”（分号）分隔，后缀和关键字用“:”（冒号）分隔

Enter ":" (colon) at the front of the search box to enter the command mode=在搜索框前面输入“：”（冒号）以进入命令模式
There are built-in commands, you can also add custom commands in the settings=下面有内置命令，也可以在设置中添加自定义命令
