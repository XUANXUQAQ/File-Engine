Welcome to the tutorial of File-Engine=Bienvenue au tutoriel du moteur de fichiers
The default Ctrl + Alt + K calls out the search bar, which can be changed in the settings.=Ctrl + Alt + k par défaut appelle la barre de recherche et peut être modifiée dans les paramètres.
You can enter the keywords you want to search here=Vous pouvez saisir les mots clés à rechercher ici
Let's see an example=Voyons un exemple.
When you enter "test" in the search bar=Lorsque vous saisissez "test" dans la barre de recherche
files with "test" in the name will be displayed below the search bar=Les fichiers avec "test" dans le nom sont affichés sous la barre de recherche
If you know multiple keywords of a file=Si vous connaissez plusieurs mots clés pour un fichier
(for example, the file name contains both "file" and "test")=(par exemple, le nom du fichier contient à la fois "fichier" et "Test")
you can separate them with ";" (semicolon) to search together as keywords.=Vous pouvez les séparer par ";" (point - virgule) et les rechercher ensemble comme mots clés.
When entering "/test" in the search bar=Lorsque vous saisissez "/test" dans la barre de recherche
the file containing "test" in the path will be displayed below the search bar=Les fichiers avec "Test" dans le chemin sont affichés sous la barre de recherche
Add "::" + suffix after the keyword to achieve a more precise search=Ajoutez "::" + suffixe après le mot-clé pour obtenir une recherche plus précise
The program has the following four suffixes=Ce programme a les quatre suffixes suivants
not case sensitive=Insensible à la casse
"::d" is the suffix for searching only folders="::d" est le suffixe pour rechercher uniquement les dossiers
"::f" is the suffix to search only for files="::f" est le suffixe pour rechercher uniquement les fichiers
"::full" means full word matching, but case insensitive="::full" signifie une correspondance complète des mots, mais insensible à la casse
"::case" means case sensitive="::case" signifie sensible à la casse
You can also combine different suffixes to use=Vous pouvez également combiner différents suffixes
Different keywords are separated by ";" (semicolon), suffix and keywords are separated by "::" (colon)=Les différents mots - clés sont séparés par ";" (point - virgule), les suffixes et les mots - clés sont séparés par "::" (deux points)

You can modify these hotkeys in the settings=Vous pouvez modifier ces hotkeys dans les paramètres
Enter ":" (colon) at the front of the search box to enter the command mode=Saisissez «: » (deux points) devant la zone de recherche pour entrer en mode commande
There are built-in commands, you can also add custom commands in the settings=Avec des commandes intégrées, vous pouvez également ajouter des commandes personnalisées à vos paramètres
If you find that some files cannot be searched, you can enter ":update" in the search bar to rebuild the index.=Si vous trouvez que certains fichiers ne peuvent pas être recherchés, vous pouvez reconstruire l'index en entrant «: update » dans la barre de recherche.
