Welcome to the tutorial of File-Engine=Добро пожаловать в курс файлового движка
The default Ctrl + Alt + K calls out the search bar, which can be changed in the settings.=По умолчанию Ctrl + Alt + K перемещает строку поиска, которая может быть изменена в настройках.
You can enter the keywords you want to search here=Здесь вы можете ввести ключ для поиска
Let's see an example=Давайте посмотрим пример
When you enter "test" in the search bar=Введите "test" в строке поиска
files with "test" in the name will be displayed below the search bar=файл с именем "test" будет показан под строкой поиска
If you know multiple keywords of a file=если вы знаете несколько ключевых слов в файле
(for example, the file name contains both "file" and "test")=(например, имя файла содержит как "file", так и "test")
you can separate them with ";" (semicolon) to search together as keywords.=Вы можете прочесть их раздельно, как ключевые слова.
When entering "/test" in the search bar=Введите "/ test" в строке поиска
the file containing "test" in the path will be displayed below the search bar=путь к файлу, содержащему "test", будет показан под строкой поиска
Add ":" + suffix after the keyword to achieve a more precise search=Добавьте ":" + суффикс после ключевого слова для более точного поиска
The program has the following four suffixes=Эта программа имеет следующие четыре суффикса
not case sensitive=писать без различия размера
":d" is the suffix for searching only folders=":d" - это суффикс для поиска только папок
":f" is the suffix to search only for files=":f" - это суффикс для поиска только файлов
":full" means full word matching, but case insensitive=":full" означает полное совпадение слов, но без учета регистра.
":case" means case sensitive=":case" означает чувствительность к регистру
You can also combine different suffixes to use=Вы также можете комбинировать различные суффиксы
Different keywords are separated by ";" (semicolon), suffix and keywords are separated by ":" (colon)=различные ключевые слова используются ";" (запятая) для разделения, суффикс и ключевые слова ":" (двоеточие) для разделения

You can modify these hotkeys in the settings=Вы можете изменить эти горячие клавиши в настройках
Enter ":" (colon) at the front of the search box to enter the command mode=Введите ":" (двоеточие) для входа в командный режим
There are built-in commands, you can also add custom commands in the settings=есть встроенные команды, которые могут быть добавлены в настройки
If you find that some files cannot be searched, you can enter ":update" in the search bar to rebuild the index.=если некоторые файлы не могут быть найдены, можно ввести «: update » в строку поиска для восстановления индекса.
