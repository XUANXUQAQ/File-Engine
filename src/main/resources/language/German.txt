Welcome to the tutorial of File-Engine=Willkommen im Tutorial von File-Engine
The default Ctrl + Alt + K calls out the search bar, which can be changed in the settings.=Die Standardeinstellung Ctrl + Alt + K ruft die Suchleiste auf, die in den Einstellungen geändert werden kann.
You can enter the keywords you want to search here=Hier können Sie die Suchbegriffe eingeben, die Sie suchen möchten
Let's see an example=Lassen Sie uns ein Beispiel sehen
When you enter "test" in the search bar=Wenn Sie "testen" in der Suchleiste eingeben
files with "test" in the name will be displayed below the search bar=Dateien mit "test" im Namen werden unter der Suchleiste angezeigt
If you know multiple keywords of a file=Wenn Sie mehrere Schlüsselwörter einer Datei kennen
(for example, the file name contains both "file" and "test")=(zum Beispiel enthält der Dateiname sowohl "Datei" als auch "test")
you can separate them with ";" (semicolon) to search together as keywords.=Sie können sie mit ";" (Semikolon) trennen, um gemeinsam als Schlüsselwörter zu suchen.
When entering "/test" in the search bar=Bei Eingabe von "/test" in die Suchleiste
the file containing "test" in the path will be displayed below the search bar=die Datei mit "test" im Pfad wird unterhalb der Suchleiste angezeigt
Add ":" + suffix after the keyword to achieve a more precise search=Fügen Sie „:“ + Suffix nach dem Schlüsselwort hinzu, um eine präzisere Suche zu erreichen
The program has the following four suffixes=Das Programm hat die folgenden vier Suffixe
not case sensitive=nicht Groß-/Kleinschreibung
":d" is the suffix for searching only folders=":d" ist das Suffix, um nur Ordner zu durchsuchen
":f" is the suffix to search only for files=":f" ist das Suffix, um nur nach Dateien zu suchen
":full" means full word matching, but case insensitive=":full" bedeutet vollständige Wortsuche, aber ohne Berücksichtigung der Groß-/Kleinschreibung
":case" means case sensitive=":case" bedeutet Groß-/Kleinschreibung
You can also combine different suffixes to use=Sie können auch verschiedene Suffixe kombinieren
Different keywords are separated by ";" (semicolon), suffix and keywords are separated by ":" (colon)=Verschiedene Schlüsselwörter werden durch ";" (Semikolon), Suffix und Schlüsselwörter durch ":" (Doppelpunkt) getrennt.

Click "Alt+ Enter" to copy the file path=Klicken Sie auf "Alt+ Enter", um den Dateipfad zu kopieren
You can modify these hotkeys in the settings=Sie können diese Hotkeys in den Einstellungen ändern
Enter ":" (colon) at the front of the search box to enter the command mode=Geben Sie ":" (Doppelpunkt) an der Vorderseite des Suchfeldes ein, um den Befehlsmodus einzugeben
There are built-in commands, you can also add custom commands in the settings=Es gibt eingebaute Befehle, Sie können auch benutzerdefinierte Befehle in den Einstellungen hinzufügen
If you find that some files cannot be searched, you can enter ":update" in the search bar to rebuild the index.=Wenn Sie feststellen, dass einige Dateien nicht durchsucht werden können, können Sie in der Suchleiste ":update" eingeben, um den Index wieder aufzubauen.
